# Forza 4 Q-Learning

Implementazione in python di un agente Q-Learning

## Problema

Creare un agente che riesca a giocare contro un giocatore vero.

## Esecuzione

Requisiti python3 e numpy, basta eseguire il file main.py.

## Limitazioni

Agente abbastanza scarso, griglia ridotta per facilitare il learning
